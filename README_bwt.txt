The Script implements Burrows Wheeler Transform on any sequence provided by the user.
It generates the Burrows Wheeler Matrix by sorting them lexicographically.
The Transform is taken by extracting the last column of matrix.
The second part of this script asks for multiple inputs by the user so that he can check whether his given sequences are present in the main sequence or not. In other words he looks for exact matches in the given sequence.
The scripts provides the position for exact matches if found.
